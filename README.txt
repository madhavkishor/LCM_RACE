# LCM Race — Interactive Maths Game (Assignment Submission)

## 📘 Overview
This project is an **interactive PHP + JavaScript game** that helps students (Grades 4–7) understand the **Least Common Multiple (LCM)** through play.  
It visualizes multiples, allows exploration, and reinforces understanding through immediate feedback.

## 🧩 Features
- Visual **Multiples Explorer**
- **Interactive choices** for LCM selection
- **Auto-highlight** of common multiples
- **Score tracking** and optional save (writes to submissions.csv)
- Responsive, clean UI suitable for classrooms

## 🧠 Teaching Use
1. Start a new challenge — two numbers are shown.
2. Students explore multiples by clicking them.
3. Discuss the pattern and concept of “common multiples.”
4. Let them choose what they believe is the LCM.
5. Reinforce through visual feedback and hint messages.

## ⚙️ Setup
1. Place `index.php` in `htdocs/lcm_race/` folder of XAMPP or WAMP.
2. Start Apache and open `http://localhost/lcm_race/`.
3. Play and explore!

## 📂 Files
- `index.php` — main interactive game
- `submissions.csv` — created automatically on saving scores

## 🧑‍🏫 Notes
This project demonstrates how **mathematical visualization** can make learning more intuitive and fun.  
It also gives a practical glimpse into **educational game development** using **PHP + JS**.
